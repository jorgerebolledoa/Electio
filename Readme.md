# 📘 Electio – Backend API

Backend REST desarrollado en **Node.js + Express** con **PostgreSQL**, diseñado para soportar el sistema de postulación a electivos *Electio*, conforme al Diseño Modular definido en el proyecto.

---

## 🧱 Arquitectura

- **Backend**: Node.js + Express  
- **Base de Datos**: PostgreSQL 16  
- **Contenedores**: Docker + Docker Compose  
- **Modelo de datos**: Relacional, normalizado, con integridad referencial  
- **Control de acceso**: Basado en roles  

---

docker compose up --build
